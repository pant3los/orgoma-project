# Σχεδιασμός και Ανάπτυξη Εφαρμογών Κινητού Υπολογισμού
# Project (ομαδική έως 3 άτομα)

Το παρόν αποθετήριο αφορά το OrgOMa (Organic Olive Management app) project και θα μεταφορτώνετε όλα τα παραδοτέα εδώ σύμφωνα με τις οδηγίες της εκφώνησης που συμπεριλαμβάνεται στο παρόν αποθετήριο.
Η επιλογή ημ/νίας και ώρας προφορικής εξέτασης γίνεται μέσω ψηφοφορίας, σύνδεσμος της οποία θα προστεθεί στο παρόν αρχείο και θα ενημερωθείτε με ανακοίνωση.
